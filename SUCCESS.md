## 🚀 Quick Start (3 Commands)

```bash
# 1. Setup (first time only - may take 5-10 minutes)
chmod +x setup.sh && ./setup.sh

# 2. Test installation
chmod +x test.sh && ./test.sh

# 3. Start the app
chmod +x start.sh && ./start.sh
```

**Then open**: http://localhost:3000 🎊

## 📖 How to Use

### For Users

1. **Open the App**: Navigate to http://localhost:3000
2. **Select Ayah**: Choose a Surah and Ayah to practice
3. **Grant Permission**: Allow microphone access
4. **Record**: Click 🎤 and recite
5. **Review**: Get instant feedback with:
   - Transcription of your recitation
   - Accuracy score (0-100)
   - Detected Tajweed rules
   - Specific corrections
   - Visual error highlighting

### For Developers

```bash
# Backend API
http://localhost:8000/docs

# Health Check
curl http://localhost:8000/health

# Test Endpoint
curl http://localhost:8000/api/quran/ayah/1/1
```

## 🔥 Advanced Features

### AI Models Available
1. **Tarteel Whisper Tiny** (150MB) - Fastest
2. **Tarteel Whisper Base** (290MB) - Balanced
3. **Tarteel Wav2Vec2** (1.2GB) - Most accurate
4. **Arabic Wav2Vec2** - General fallback
5. **OpenAI Whisper** - Multi-language support

### Tajweed Rules Covered
- Noon Sakinah & Tanween (4 rules)
- Meem Sakinah (3 rules)
- Madd/Prolongation (5 types)
- Qalqalah (Echo sound)
- Ghunna (Nasal sound)
- And more...

### Technical Stack
- **Backend**: Python, FastAPI, PyTorch, Transformers
- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **AI**: Tarteel, Whisper, Wav2Vec2
- **Communication**: WebSocket, REST API
- **Deployment**: Docker, Docker Compose
